      • setRequestedSurfaceViewForCameraDisplay

         final void setRequestedSurfaceViewForCameraDisplay(@NonNull() CameraSurface requestedSurface)

        In order to display camera preview, camera view must use one of two android's view's that arecapable of displaying camera preview video stream: android.view.SurfaceView and android.view.TextureView. By default, android.view.SurfaceView will bepreferred because it is faster and more power efficient. Sometimes, when youcreate complex UI on top of camera screen, animations or UI may render incorrectly on somedevices when using android.view.SurfaceView for displaying camera preview.In that case, you can use this method to force setting the surface you want.

        requestedSurface - Which view should be used to display camera preview.
